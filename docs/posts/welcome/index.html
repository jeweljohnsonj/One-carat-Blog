Making of one-carat-blog (second version)

Hello! my name is Jewel Johnson and I am the creator of this blog. The main reason why I made this blog was to have a source of medium for me to write articles in science and share them with the world. This first version of this blog was made using the distill package in R and you can find the link to it here. From then I moved on and remade the blog using the latest quarto package in R. This new package proves to be an upgrade from distill and I am starting to love the new features of it. Since I am also fond of using R, I took it upon me to write a few tutorials focusing on beginners, which you can find in this blog.

How did I make this blog

There are several well-explained tutorials which explain from scratch how one can build a blog. If you are planning to use quarto for making your blog, then the quarto website itself features well written beginner friendly tutorials. Apart from that I found two other posts which also proved very useful while creating this blog. The links are given below.

  • Quarto website
  • Porting from distill to quarto
  • We don’t talk about Quarto

My great plan

Since I will be applying for a PhD, I hope I can showcase my skills using this blog and make myself standout from the rest. Last time when I first made the blog, the initial enthusiasm died out after one month and I did not write any posts after it. So this time, to avoid that, I took it to myself to write at lest one post per month and get the blog going. with that said I welcome you reader to my blog and I hope you enjoy it with as much as fun that I had while making it.
