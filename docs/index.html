Welcome to One Carat Blog

My name is Jewel Johnson and I am the creator of this blog. The blog was first made using the {distill} package in R. Eventually the blog was remade using the new {quarto} package. I write science and R-programming articles. I have also made a tutorial on data visualization and manipulation using R, and an intro to basic statistics using R, so do check them out!
